# Animation UI Motion

Use this for app/web interface motion: button interactions, screen transitions, onboarding motion, loading states, cards expanding/collapsing, hover/tap behavior, motion specs, and making UI feel less stiff.

Prefer `ui-ux__motion-design`, `ui-ux__ui-motion-wiki`, and `creative__claude-design`. If implementation is requested, inspect the actual app/web code and verify on the target surface rather than only describing motion principles.

## Output standard

For quick explorations, produce a short MP4 or GIF plus a still/contact sheet when useful. For polished work, render a clean MP4 for the requested surface and verify it with `ffprobe` before calling it done.

## Naming note

Telegram shows this command with underscores because Telegram slash commands cannot contain hyphens. Hermes resolves the hyphenated skill name normally elsewhere.
