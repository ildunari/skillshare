# GitHub Research Playbook

## Goals
- Discover candidates (Trending + search)
- Evaluate quality and maintenance
- Compare alternatives

## Signals
- Stars, forks, issues, PR velocity
- Release cadence, tags, semantic versioning
- CI status, test coverage, SPM support
- Ownership distribution (bus factor)

- scripts/github_trends_analyzer.py
- scripts/repo_analyzer.py
