# Visual Specs

## Preferred visuals

### 1. Canonical source → target topology map
Use for: source-of-truth and sync questions.

### 2. Distribution drift matrix
Use for: runtime target sync status.

### 3. Skill family map
Use for: duplicates, near-duplicates, supersedes chains.

### 4. Delete/archive ladder
Use for: cleanup planning.

### 5. Canonical library health board
Use for: `~/.config/skillshare/skills` audits.

## Reporting rule
When the question is about the canonical library, show source/target topology and distribution matrix first. When the question is about overlap or cleanup, show skill families and delete ladder first.
