## Originality rule

Prefer original characters and visual traits over named copyrighted characters or living artists. When the user names a protected style or character, capture broad traits: whimsical watercolor fantasy, soft hand-painted backgrounds, expressive simple characters, cinematic manga paneling, retro comic halftone, etc.

## Children's book watercolor

Template:

```text
Create a children's book illustration of [scene].
Character: [original character description].
Style: warm hand-painted watercolor, soft outlines, gentle expressions, textured paper, cozy palette.
Composition: clear focal point, picture-book proportions, enough background detail to tell the story.
Constraints: original character, no text unless requested, no scary details, no watermark.
```

## Manga page

Template:

```text
Create a black-and-white manga page with [number] panels.
Story beat: [scene].
Panel plan: [panel descriptions].
Style: cinematic manga paneling, expressive faces, strong inks, speed lines or screentone only where useful.
Text: [exact dialogue/captions] or no text.
Constraints: original characters, no copyrighted series references, no extra speech bubbles, readable panel flow.
```

## Western comic page

Template:

```text
Create a full-color comic page with [number] panels.
Characters: [original characters].
Story: [sequence].
Style: bold ink outlines, dynamic panel composition, controlled color palette, readable expressions.
Text: [exact captions/dialogue] or no text.
Constraints: no copyrighted characters, no extra text, no cluttered panels.
```

## Storyboard

Template:

```text
Create a storyboard sheet for [scene/video/ad].
Grid: [number] panels with clean borders.
Each panel: [shot list].
Style: grayscale storyboard sketch, clear camera framing, readable action, minimal shading.
Constraints: no extra shots, no polished final art, no tiny text.
```

## Anime character sheet

Template:

```text
Create an original anime-style character reference sheet for [character].
Include: front view, side view, back view, three expressions, key prop, small notes panel.
Character details: [hair, outfit, colors, personality].
Style: clean character-design sheet, crisp line art, readable shapes, consistent proportions.
Constraints: original character, no copyrighted costume, no extra characters, no watermark.
```

## Character consistency workflow

Use a two-step workflow for recurring characters:

1. Generate a character anchor sheet: front/side/back, expressions, outfit, palette, props.
2. Use the anchor as a reference for subsequent scenes and repeat locked details in the prompt.

Continuation prompt:

```text
Continue using the same character from the reference image.
Preserve: facial features, proportions, hair, outfit, color palette, personality, and key prop.
New scene: [scene/action].
Style: same illustration style, same line quality, same color treatment.
Constraints: do not redesign the character, do not change age/body type/outfit unless requested.
```

## Fantasy or sci-fi concept art

Template:

```text
Create concept art for [character/environment/object].
World: [setting rules].
Design language: [materials, motifs, silhouette, palette].
Composition: [hero view / turnaround / environment key art].
Mood: [tone].
Constraints: original design, no copied franchise elements, no fake logos, no excessive detail that hides silhouette.
```

## Editorial illustration

Template:

```text
Create an editorial illustration for an article about [topic].
Metaphor: [visual metaphor].
Style: sophisticated magazine illustration, limited palette, clean composition, strong symbolic clarity.
Audience: [publication/audience].
Constraints: no text unless requested, no obvious stock imagery, no clutter.
```

## Line art and coloring page

Template:

```text
Create clean black-and-white line art of [subject] for [purpose].
Style: smooth confident outlines, closed shapes suitable for coloring, white background, no shading or grayscale fill.
Composition: centered, readable, age-appropriate detail level.
Constraints: no text, no watermarks, no tiny texture, no broken outlines.
```

## Collage and mixed media

Template:

```text
Create a mixed-media collage illustration about [theme].
Materials: torn paper, ink marks, photographic texture, printed halftone, layered shapes.
Composition: [layout], clear focal point, balanced negative space.
Palette: [colors].
Constraints: original imagery, no real logos, no unreadable text fragments unless requested.
```
