## Print rule

For print-like outputs, specify trim, safe margins, bleed guides, and exact text only if you need mockup-level accuracy. For final production, rebuild typography in design software after using image generation for concept art.

## Magazine spread

Template:

```text
Create a two-page editorial magazine spread about [topic].
Layout: strong headline, dek/subhead, image area, pull quote, small sidebar, generous margins.
Exact visible text: [short headline/subhead/pull quote].
Style: [luxury/editorial/science magazine/culture journal], sophisticated typography, balanced grid, high readability.
Constraints: no lorem ipsum, no fake long paragraphs, no extra headlines, no watermark.
```

## Book cover

Template:

```text
Create a book cover design for an original book titled "[title]" by "[author]".
Genre: [genre].
Visual concept: [concept].
Typography: [serif/sans/display], title large and readable, author smaller.
Style: [literary/minimal/pulp/fantasy/nonfiction], print-ready composition.
Constraints: no existing franchise imagery, no fake publisher logos, no extra text, no watermark.
```

## Typography poster

Template:

```text
Create a typography-led poster.
Exact text, render once and verbatim: "[text]".
Style: [Bauhaus/Swiss/Art Deco/brutalist/editorial], strong hierarchy, high contrast, geometric composition.
Canvas: [portrait/landscape].
Constraints: no extra words, no misspellings, no tiny text, no watermark.
```

## Brochure or flyer

Template:

```text
Create a polished [brochure/flyer] for [event/product/place].
Format: [one-page flyer / tri-fold brochure mockup].
Exact text blocks: [short text].
Visuals: [photos/illustrations/icons].
Style: [premium hospitality / community event / tech conference], clear hierarchy, readable typography.
Constraints: no invented contact info, no extra text, no real logos unless provided.
```

## Bookmark, card, and small print

Template:

```text
Create a print-ready concept mockup for a [bookmark/greeting card/invite].
Canvas: [size/aspect]. Include trim, bleed, and safe margin guides as subtle production marks.
Artwork: [concept].
Exact text: "[text]".
Style: [Art Deco/minimal/watercolor/editorial], high print-quality composition.
Constraints: no extra text, no watermark, no real trademarks.
```

## Handwritten notebook page

Template:

```text
Create a realistic handwritten notebook page about [topic].
Paper: lined notebook paper, natural pencil or pen texture, slight page imperfections.
Content: [short exact notes or bullet labels].
Style: believable student notes, readable handwriting, small diagrams, natural spacing.
Constraints: no invented facts, no illegible scribbles, no extra text beyond the notes.
```

## Multilingual lettering

Template:

```text
Create a multilingual typography composition featuring these exact scripts/text snippets: [list].
Style: [editorial poster / cultural brochure / UI localization mockup].
Layout: balanced hierarchy, each script readable, strong contrast, no invented characters.
Constraints: preserve the exact text, do not translate unless asked, no extra words.
```

## Wayfinding and signage

Template:

```text
Create a wayfinding signage system concept for [place].
Include: directional signs, map panel, room labels, icon set.
Exact labels: [list].
Style: clean environmental graphic design, accessible contrast, readable from distance, consistent icons.
Constraints: no fake locations, no extra labels, no tiny text.
```

## Zine or risograph layout

Template:

```text
Create a zine-style page about [topic].
Style: risograph print texture, limited spot colors, imperfect registration, collage blocks, punk/editorial energy.
Layout: [cover/page/spread], clear focal point.
Text: [short exact text] or no text.
Constraints: no unreadable filler, no real logos, no watermark.
```
