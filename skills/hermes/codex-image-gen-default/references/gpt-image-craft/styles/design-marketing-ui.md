## Creative brief rule

Marketing and UI prompts work best when written like professional briefs. Specify brand, audience, channel, exact copy, visual hierarchy, placement, and what not to invent. Avoid generic “make it modern” prompts.

## Advertising campaign image

Template:

```text
Create a polished campaign image for [brand/product].
Brand: [positioning, audience, personality].
Scene/concept: [scene].
Exact copy, render once and verbatim: "[tagline]".
Composition: [placement of product, people, headline, negative space].
Style: [premium editorial / streetwear / warm lifestyle / tech minimal].
Constraints: original non-infringing brand, no real logos, no extra text, no watermark.
```

## Logo and brand mark

Logo generation benefits from simplicity and scalability. Ask for original, non-infringing marks with a strong silhouette and negative space. Do not ask for a copy of an existing logo.

Template:

```text
Create an original logo mark for [company/product], a [description].
Brand personality: [traits].
Use case: must read at app-icon size and on packaging.
Style: clean vector-like mark, strong silhouette, balanced negative space, simple geometry, flat color.
Deliverable: one centered logo on a plain background with generous padding.
Constraints: no existing logos, no trademarks, no gradients unless essential, no watermark.
```

## Brand mood board

Template:

```text
Create a polished brand mood board for [brand/concept].
Include: color palette swatches, typography samples, product mockup, texture samples, photography direction, icon style.
Style: premium design-studio presentation, clean grid, readable labels, cohesive visual system.
Exact words to include: [brand name, short labels].
Constraints: no fake external logos, no unreadable filler text, no clutter.
```

## Packaging design

Template:

```text
Create a realistic packaging design mockup for [product].
Package type: [box/pouch/can/bottle/blister].
Brand direction: [style].
Exact front label text: "[text]".
Visual hierarchy: product name largest, flavor/variant secondary, small callout [callout].
Rendering: realistic studio product photography, crisp label printing, accurate material texture.
Constraints: original design, no real brand logos, no extra claims, no watermark.
```

## UI mockup

UI prompts should describe the product as if it already exists. Avoid “concept art” language unless you want a fantasy interface.

Template:

```text
Create a realistic [mobile/web] UI mockup for [product].
User: [audience].
Screen: [specific screen].
Layout: [navigation, cards, lists, charts, buttons].
Copy labels: [short exact UI labels].
Style: usable production interface, clean spacing, accessible contrast, modern sans-serif, restrained accents.
Constraints: no lorem ipsum, no tiny unreadable text, no fake logos, no decorative clutter.
```

## Landing page hero

Template:

```text
Create a landing-page hero section mockup for [product/service].
Canvas: wide 16:9.
Hero copy: "[headline]" and "[subhead]".
Layout: [left copy + right visual / centered / split screen].
Visual: [product shot/abstract system/illustration].
Style: polished startup website, clear CTA buttons, generous spacing, accessible typography.
Constraints: no lorem ipsum, no fake nav clutter, no invented logos.
```

## Social carousel

Template:

```text
Create a 5-card social carousel design for [platform/topic].
Cards: 1 cover, 2 key point, 3 example, 4 checklist, 5 call to action.
Exact short text per card: [list].
Style: cohesive editorial social design, bold hierarchy, consistent grid, mobile-readable text.
Constraints: no extra text, no tiny paragraphs, no watermark, no off-brand colors.
```

## Mini thumbnail

Best for YouTube thumbnails, small preview cards, app cards, marketplace tiles, and social previews.

Prompt levers:

- large subject silhouette;
- very short text, ideally under 4 words;
- high contrast;
- readable at small size;
- no clutter.

Template:

```text
Create a high-impact mini thumbnail for [video/post/product].
Canvas: 16:9 landscape.
Subject: [large focal subject] occupying most of the frame.
Text, render once and verbatim: "[1-4 words]".
Style: bold, clean, high contrast, readable at small size, strong silhouette, simple background.
Constraints: no tiny text, no clutter, no extra words, no watermark.
```

## Pitch deck slide

Template:

```text
Create one polished 16:9 pitch deck slide titled "[title]".
Audience: investors/executives.
Main message: [one sentence].
Content blocks: [chart/diagram/KPI cards].
Data and labels: [exact values].
Style: clean startup deck, white background, modern sans-serif, clear hierarchy, subtle grid.
Constraints: no stock photography, no clip art, no fake citations, no tiny labels.
```

## Merch concept

Template:

```text
Create a realistic merch concept mockup for [original character/brand].
Product: [t-shirt/hoodie/keychain/plush/sticker pack].
Design: [visual concept].
Presentation: premium product photography, realistic materials, clean packaging, crisp print details.
Exact packaging text: "[text]".
Constraints: original design only, no trademarks, no real brand logos, no watermark.
```
