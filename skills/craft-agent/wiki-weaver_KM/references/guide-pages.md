## How to Update

1. Read the guide page
2. Find the appropriate section (or "Recent Additions" if it exists)
3. Check if the note is already listed (dedup)
4. If not listed, append: `- [[note-path]] — One-line description`
5. Never modify Dataview query blocks (```dataview ... ```)
6. Never restructure existing sections
