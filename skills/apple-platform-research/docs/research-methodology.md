# Research Methodology

This document provides the detailed, step-by-step research process used by the Skill.

## Phases

1. Intent framing
2. Source mapping (primary vs secondary)
3. Signal collection
4. Evidence weighting
5. Synthesis
6. Action extraction
7. Review & iterate

## Tactics

- Start from primary sources (Apple docs, headers, release notes, WWDC)
- Triangulate claims across at least two sources
- Keep an evidence table per question
- Prefer reproducible examples and code
- Version-control notes and artifacts

## Outputs

- Research brief (1-2 pages)
- Detailed report (with citations)
- Risk register
- Migration plan and checklists
