# Source Evaluation

Use this checklist to score sources: Authority, Recency, Reproducibility, Consistency, Bias, License.

**Authority**: Apple docs, maintainers, senior practitioners.
**Recency**: Aligned with current Xcode/OS/Swift versions.
**Reproducibility**: Code builds; steps complete.
**Consistency**: Aligns with headers and observed behavior.
**Bias**: Commercial incentives; advocacy; partial benchmarks.
**License**: Usage terms clear for code/assets.
