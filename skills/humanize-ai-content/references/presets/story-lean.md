# Preset: story-lean

Use when the user wants a human narrative feel: personal posts, product stories, founder notes, reflective updates.

## Style

- Narrative flow with one concrete example.
- Mix sentence lengths. Use occasional short lines for emphasis.
- Keep it grounded. No invented facts.
- End cleanly without a recap paragraph.

## Signature moves

- Open with a moment or a specific detail from the source (not a thesis).
- Use one concrete example to carry the point (don’t stack three examples).
- End mid-action or mid-thought — no bow-tying.

## Do

- Keep the author’s voice (slang, humor, sharpness) if it’s in the input.
- Use sensory detail only if it exists in the source (don’t invent).
- Use natural dialogue or quotes only if the source already contains them.

## Don’t

- Don’t add new events, places, or outcomes.
- Don’t turn it into a motivational poster.
- Don’t force a “lesson learned” wrap-up paragraph.

## Anti-patterns to avoid

- Five-paragraph essay structure (intro/body/conclusion).
- Tricolon spam (“X, Y, and Z” every paragraph).
- Overusing em dashes as glue.
