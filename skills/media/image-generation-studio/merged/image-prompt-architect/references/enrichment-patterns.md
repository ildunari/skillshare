# Enrichment Patterns — Subject-Specific Micro-Prompters

> When your prompt's subject needs more specificity, use these templates to add detail without bloat. Each pattern lists the key attributes that make that subject type feel real and intentional. Pick the relevant attributes — don't use all of them every time.

## How to Use

These are **detail injection templates**, not standalone prompts. When your main prompt has a subject that could be more specific, pull the relevant attributes from the matching pattern and weave them into your narrative.

**Before enrichment:** "A woman in a coffee shop"
**After enrichment:** "A woman in her late 30s with a warm complexion, wearing a cable-knit charcoal sweater and tortoiseshell reading glasses, sitting at a reclaimed wood table in a specialty coffee shop, hands wrapped around a ceramic latte cup, expression is 'just read something interesting' — slight eyebrow raise, faint smile"

## Person / Character

| Attribute | Examples |
|---|---|
| **Age cues** | Early 20s, late 40s, silver-haired, youthful freckles |
| **Build/physique** | Mesomorphic, lean, broad-shouldered, petite, athletic |
| **Skin/complexion** | Warm olive, deep brown, freckled fair, sun-weathered |
| **Hair** | Cropped undercut, long braids, slicked back, natural curls |
| **Clothing/texture** | Cable-knit wool, distressed denim, tailored linen, leather bomber |
| **Accessories** | Tortoiseshell glasses, chunky watch, delicate chain necklace, canvas tote |
| **Expression direction** | "Just heard good news" > "happy"; "concentrating on a difficult problem" > "focused" |
| **Pose vocabulary** | Contrapposto, three-quarter stance, hands in pockets, leaning on railing, mid-stride |
| **Inclusive representation** | Specify diversity when contextually appropriate; avoid tokenism |

**Anti-pattern:** Don't describe race/ethnicity as the primary identifier. Lead with distinctive features, clothing, and expression.

## Landscape / Environment

| Attribute | Examples |
|---|---|
| **Landform** | Jagged granite peaks, rolling chalk hills, flat salt flats, coastal cliffs |
| **Vegetation** | Wildflower meadow, dense pine canopy, scrub brush, moss-covered boulders |
| **Weather/atmosphere** | Low fog wisping, rain streaks, dust haze, crisp clear, overcast diffuse |
| **Time of day** | Pre-dawn violet, golden hour amber, harsh noon, blue hour, starlit |
| **Water** | Mirror-still lake, white-water rapids, gentle surf, frozen stream |
| **Aerial depth** | Near foreground detail → mid-ground subject → atmospheric fade to background |
| **Season** | Fresh spring green, peak autumn reds, winter bare branches, summer fullness |

## Product

| Attribute | Examples |
|---|---|
| **Material/finish** | Brushed aluminum, matte ceramic, frosted glass, soft-touch rubber, walnut veneer |
| **Manufacturing detail** | Visible seam lines, injection mold marks, CNC milling patterns, hand-stitched thread |
| **Honest imperfections** | Hairline scratches on metal, slight grain variation in wood, micro-bubbles in glass |
| **Scale cue** | Next to a hand, on a desk with known objects, with a coin for scale |
| **Hero feature** | Which specific detail should draw the eye — the clasp, the screen, the texture |
| **Surface interaction** | Grounded shadow, reflection on glossy surface, contact point with table |
| **Packaging/context** | Unboxing, on retail shelf, lifestyle in-use, isolated studio |

## Building / Architecture

| Attribute | Examples |
|---|---|
| **Architectural style** | Brutalist concrete, Art Deco ornamentation, minimalist glass-and-steel, Moorish arches |
| **Materials** | Exposed aggregate concrete, aged red brick, oxidized copper cladding, marble facing |
| **Weathering/age** | Rain stains on concrete, patinated bronze, cracked stucco, pristine new construction |
| **Structural details** | Flying buttresses, cantilever, post-and-beam, glass curtain wall, vaulted ceiling |
| **Site context** | Urban street frontage, hilltop isolated, waterfront, surrounded by old-growth trees |
| **Scale indicators** | Human figure, vehicles, surrounding buildings, trees for relative size |
| **Light interaction** | Sun casting through windows, shadow patterns from louvers, glowing interior at dusk |

## Animal

| Attribute | Examples |
|---|---|
| **Species precision** | "Bengal tiger" not "tiger"; "Blue Jay" not "bird"; "Maine Coon" not "cat" |
| **Fur/feather/scale texture** | Thick double coat, iridescent plumage, smooth scales, wiry terrier coat |
| **Pose suggesting personality** | Alert and watchful, lazy stretch, playful pounce-ready, dignified sit |
| **Environment interaction** | Perched on branch, wading in shallows, curled on sun-warmed stone, running through grass |
| **Eye detail** | Golden iris, vertical slit pupil, bright inquisitive, side-eye suspicion |
| **Markings** | Tabby stripes, dalmatian spots, raccoon mask, gradient color from head to tail |

## Food

| Attribute | Examples |
|---|---|
| **Cuisine identification** | Japanese kaiseki, rustic Italian, French patisserie, Southern American comfort |
| **Cooking method visible** | Char marks from grill, glossy reduction glaze, crispy fried crust, smoke wisps |
| **Garnish/finishing** | Microgreens, dusted powdered sugar, drizzled sauce, flaky sea salt, edible flowers |
| **Temperature cues** | Rising steam, condensation on glass, melting butter, frost on frozen dessert |
| **Plating style** | Minimalist center plate, rustic scattered, stacked architectural, family-style sharing |
| **Surface/setting** | Dark slate, white porcelain, wooden cutting board, marble counter, woven placemat |

## Abstract / Concept

For non-representational subjects (brand backgrounds, textures, mood pieces):

| Attribute | Examples |
|---|---|
| **Form language** | Flowing ribbons, geometric shards, organic blobs, crystalline structures |
| **Movement** | Diagonal sweep, radial burst, gentle drift, turbulent swirl |
| **Texture** | Glassy smooth, granular noise, fabric-like folds, liquid metallic |
| **Depth** | Layered planes, shallow DOF focus point, atmospheric depth fade |
| **Palette strategy** | Monochrome gradient, complementary tension, analogous harmony, triadic vibrance |
| **Negative space** | Empty center for text overlay, breathing room around focal element |

## UI Component (for design asset generation)

For generating images that will become UI elements:

| Attribute | Examples |
|---|---|
| **Visual weight** | Heavy/bold for CTAs, light/subtle for secondary, balanced for cards |
| **Shape language** | Rounded/friendly, angular/technical, organic/natural, geometric/precise |
| **State representation** | Default, hover, active, disabled, loading, error, success |
| **Scalability** | Must read at 16px (favicon) through 512px (hero) — specify minimum readable size |
| **Consistency markers** | Same stroke weight, corner radius, shadow depth, color treatment across set |
| **Isolation** | Clean edges, no bleed, designed for cropping/masking, transparent-background-ready |
