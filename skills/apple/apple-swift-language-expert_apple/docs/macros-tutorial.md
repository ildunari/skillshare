# macros-tutorial.md

- Macros in Swift 5.9+: freestanding (`#`) and attached (`@`) kinds.
- Build a macro target depending on SwiftSyntax; test with MacroTesting.
- Use the provided `scripts/macro_generator.py` to scaffold a package.
- Performance & DX: prefer prebuilt SwiftSyntax (Xcode 16.4+) and pin versions.
